## üîç HOW TO USE THESE SOURCES

### Step-by-Step Research Process:

#### 1. Visit Official Site (5 min)
- Go to state's .gov labor site
- Find "Employers" or "Employment Law" section
- Locate employer handbook/guide

#### 2. Identify Key Topics (10 min)
Look for these sections:
- [ ] Non-compete/Non-solicitation
- [ ] Wage & Hour Laws
- [ ] Background Check Rules
- [ ] Pay Transparency
- [ ] Paid Leave (sick/family)
- [ ] Exempt Salary Thresholds
- [ ] Drug Testing Policies

#### 3. Extract Laws (30 min)
For each law found:
- Copy exact statute citation
- Read official explanation
- Note effective date
- Save source URL
- Determine severity (error/warning/info)

#### 4. Fill Research Template (15 min)
Use `tools/state_research_template.txt`
Fill in all sections with official data

#### 5. Create JSON (10 min)
Use `tools/doc_to_state_json.py` OR manually create

#### 6. Verify (5 min)
- Double-check statute citations
- Confirm effective dates
- Ensure .gov source URLs

**Total: ~75 minutes per state**
